Thrifty Thumbs

Thrifty Thumbs

Welcome to my portfolio showcasing the creation of Thrifty Thumbs, an Android expense tracker app that goes beyond simple record-keeping. Crafted with React Native, leveraging various React Native libraries for graphs and visualization, and utilizing MySQLite for database management, Thrifty Thumbs empowers users to manage their finances efficiently, offering both expense tracking and budgeting functionalities.


Project Overview:

Thrifty Thumbs is more than just an expense tracker; it's a financial management companion designed to provide users with valuable insights into their spending habits. By harnessing the capabilities of React Native and integrating powerful libraries for data visualization, I've developed an app that not only simplifies expense tracking but also helps users plan and stick to their budgets.


Key Features:

  1. User-Friendly Interface: Thrifty Thumbs boasts an intuitive and user-friendly interface, making it easy for users to record their expenses, set budgets, and navigate through the app effortlessly.

  2. Expense Tracking: Users can efficiently log their daily expenses, categorize them, and add details such as date, merchant, and payment method. This feature ensures that users have a clear overview of their spending.

  3. Budget Management: Thrifty Thumbs empowers users to create budgets based on income, set spending limits for different categories, and track their progress throughout the month.

  4. Graphs and Visualization: The app utilizes React Native libraries for dynamic graphs and visualizations, providing users with insightful data representations of their financial habits. This enhances their ability to make informed decisions.

  5. Database Management with MySQLite: MySQLite is seamlessly integrated into the app to efficiently store and retrieve financial data, ensuring data integrity and accessibility.

Technical Insight:

Framework: React Native for cross-platform app development.

Database: SQLite for efficient data management.

React Native Libraries: Utilized various libraries for graphs, charts, and data visualization to enhance user understanding of financial data.
